# Kernfunktionen

Diese Seite behandelt die Kernfunktionalität für die App-Paket-Migration. Für die Migration von Daten innerhalb von Apps siehe [Datenmigration](/de/datamigrae/baseinfo).

## Apps in den externen Speicher migrieren

1. Wählen Sie eine einzelne App aus oder halten Sie lange gedrückt und ziehen Sie, um mehrere Apps auszuwählen (oder Cmd+Klick zum Auswählen von Apps)
2. Klicken Sie unten auf die Schaltfläche „Auf extern migrieren"
3. Warten Sie, bis die Migration abgeschlossen ist

![](https://file.shimoko.com/d/openlist/openlist/%E7%BD%91%E7%AB%99%E5%AA%92%E4%BD%93%E6%96%87%E4%BB%B6/gonomoral.gif?sign=tzb_Y7YRR9uTc4uunpWXJWYs8dOXLEa0S0AQH-qcB4I=:0)
::: tip Hinweis
Bei 100% Migrationsfortschritt kann es eine 1-2 Sekunden Pause geben, während lokale Links erstellt werden
:::

## Externe Apps zurück in den lokalen Speicher verschieben
1. Wählen Sie die App in der Externe-Apps-Bibliothek aus
2. Klicken Sie auf „Zurück in den lokalen Speicher verschieben"
![](https://file.shimoko.com/d/openlist/openlist/%E7%BD%91%E7%AB%99%E5%AA%92%E4%BD%93%E6%96%87%E4%BB%B6/qianhui.gif?sign=VDEq_lQvfhnb1vhUljwh3PVhNFkwjpKcRAeQ-Ht5Mnc=:0)

## Externe Apps lokal verknüpfen
1. Wählen Sie die App in der Externe-Apps-Bibliothek aus
2. Klicken Sie auf „Lokal verknüpfen"
![](https://file.shimoko.com/d/openlist/openlist/%E7%BD%91%E7%AB%99%E5%AA%92%E4%BD%93%E6%96%87%E4%BB%B6/linkback.gif?sign=3xdVe7IFYcMfwNt1K9WMk8ZQYIlNMWngjZoTkNwA610=:0)

## Verknüpfung aufheben
Klicken Sie auf die Schaltfläche „Verknüpfung aufheben" auf der rechten Seite der verknüpften Apps-Liste.
![](https://file.shimoko.com/d/openlist/openlist/%E7%BD%91%E7%AB%99%E5%AA%92%E4%BD%93%E6%96%87%E4%BB%B6/jiechu.gif?sign=CLDMEQxHts1S2mBEnka9HUYT2XMzcnBwX1Sjfas9Dho=:0)

# 快速开始
## 安装AppPorts
安装 AppPorts 需要以下两个前置需求   
1. 拥有一个稳定的外部存储媒介（如硬盘）
2. 操作系统不低于 macOS 12.0 (Monterey) 或更高版本

### 下载
前往 [Github releases](https://github.com/wzh4869/AppPorts/releases)页下载最新版本 dmg 安装包

::: tip
如此链接无法打开，请访问此链接来获取安装包[直链网盘](https://file.shimoko.com/AppPorts)
:::   

![](https://file.shimoko.com/d/openlist/openlist/%E7%BD%91%E7%AB%99%E5%AA%92%E4%BD%93%E6%96%87%E4%BB%B6/download.gif?sign=Xb9FOEqPxR8Q7WLixKzg5NCYcjVzmzq2eh0634xGdG0=:0)


### 安装并启动
1. 打开 .dmg 安装包  
2. 将应用拖动到 Application 文件夹中
3. 启动应用

![](https://file.shimoko.com/d/openlist/openlist/%E7%BD%91%E7%AB%99%E5%AA%92%E4%BD%93%E6%96%87%E4%BB%B6/install.gif?sign=dg-gU67tz19m6DGdI3NywEAcuqKnyTpWGas0YhZeGfM=:0)


### 必要授权
首次运行时，AppPorts 需要完全磁盘访问权限来读取和修改 /Applications 目录。
1. 打开系统设置 → 隐私与安全性。
选择完全磁盘访问权限。  
2. 点击 + 按钮，添加 AppPorts，然后打开开关。   
3. 重新启动 AppPorts。   

![](https://file.shimoko.com/d/openlist/openlist/%E7%BD%91%E7%AB%99%E5%AA%92%E4%BD%93%E6%96%87%E4%BB%B6/outh.gif?sign=fTXqbKCR_tZBKDb6p1DziuJYjD9NZAJk-Zsw7c4oOJM=:0)
   
#### AppStore 应用自更新授权
macOS 15.1（Sequoia） 及以上操作系统的用户，需在 AppStore 中开启 “下载并安装大型 App 到独立存储盘“ 来确保 AppPorts 创建外部存储的/application 文件夹以支持对 AppStore 应用自动更新 
::: warning ⚠️ macOS 15.1（Sequoia） 的之前版本系统因操作系统限制暂不支持该功能
需在 AppPorts 右上角设置-勾选开启来自 AppStore 应用的迁移功能。后续应用更新需手动二次迁移以覆盖。
:::    
   
1. 打开 AppStore
2. 在状态栏中点击设置-勾选“下载并安装大型 App 到独立存储盘“并选择与 AppPorts 中外部存储库相同的外置存储媒介

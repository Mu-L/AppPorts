# 外部存儲指南

## 推薦配置

| 配置項 | 推薦值 | 說明 |
|--------|--------|------|
| 容量 | 256 GB 及以上 | 視遷移應用數量而定 |
| 接口 | USB 3.0 及以上 / Thunderbolt | USB 2.0 速度較慢，大型應用遷移耗時較長 |
| 文件系統 | APFS | 支持克隆、快照，性能最佳 |

## 接口性能對比

| 接口 | 理論速度 | 實際遷移速度 | 適用場景 |
|------|----------|-------------|----------|
| USB 2.0 | 480 Mbps | ~30 MB/s | 不推薦，速度過慢 |
| USB 3.0 (USB-A) | 5 Gbps | ~350 MB/s | 基本夠用 |
| USB 3.1 Gen 2 (USB-C) | 10 Gbps | ~700 MB/s | 推薦 |
| Thunderbolt 3/4 | 40 Gbps | ~2500 MB/s | 最佳性能 |
| NVMe (Thunderbolt) | 40 Gbps | ~2800 MB/s | 最佳性能 |

## 文件系統建議

### APFS（推薦）

- 支持克隆、快照、空間共享
- 性能最佳，尤其適合 SSD
- macOS 原生支持

### HFS+

- 兼容性好，適合舊款 Mac
- 不支持克隆和快照
- 適合機械硬盤

### exFAT

- 跨平臺兼容（macOS + Windows）
- 不支持硬鏈接和克隆
- 性能相對較低
- 適合需要在多系統間使用的場景

## 容量規劃

AppPorts 遷移後的外部存儲使用量取決於遷移的應用和數據目錄大小。以下爲常見應用的參考大小：

| 應用類型 | 大小 |
|----------|------|
| Chrome | ~500 MB |
| Microsoft Office | ~5 GB |
| Adobe Creative Cloud | ~20-50 GB |
| Xcode | ~15 GB |
| Final Cut Pro | ~5 GB |
| 本地大語言模型（Ollama） | ~4-30 GB |

::: tip 💡 容量建議
- 輕度使用（5-10 個應用）：128 GB
- 中度使用（10-20 個應用）：256 GB
- 重度使用（20+ 應用 + 數據目錄）：512 GB 及以上
:::

## 注意事項

- 外部存儲需保持連接，已遷移的應用和數據目錄在離線時無法使用
- 建議定期備份外部存儲上的數據
- 避免在遷移過程中拔出外部存儲
- 如果外部存儲出現故障，可通過 AppPorts 將應用遷回本地

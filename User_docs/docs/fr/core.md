# Fonctionnalités principales

Cette page couvre les fonctionnalités de base pour la migration des packages d'applications. Pour la migration des données dans les applications, voir [Migration des données](/fr/datamigrae/baseinfo).

## Migrer des applications vers le stockage externe

1. Sélectionner une application unique ou appuyer longuement et glisser pour sélectionner plusieurs applications (ou Cmd+clic pour sélectionner des applications)
2. Cliquer sur le bouton « Migrer vers l'externe » en bas
3. Attendre la fin de la migration

![](https://file.shimoko.com/d/openlist/openlist/%E7%BD%91%E7%AB%99%E5%AA%92%E4%BD%93%E6%96%87%E4%BB%B6/gonomoral.gif?sign=tzb_Y7YRR9uTc4uunpWXJWYs8dOXLEa0S0AQH-qcB4I=:0)
::: tip Note
À 100% de progression de la migration, il peut y avoir une pause de 1-2 secondes lors de la création des liens locaux
:::

## Déplacer les applications externes vers le local
1. Sélectionner l'application dans la bibliothèque Applications externes
2. Cliquer sur « Déplacer vers le local »
![](https://file.shimoko.com/d/openlist/openlist/%E7%BD%91%E7%AB%99%E5%AA%92%E4%BD%93%E6%96%87%E4%BB%B6/qianhui.gif?sign=VDEq_lQvfhnb1vhUljwh3PVhNFkwjpKcRAeQ-Ht5Mnc=:0)

## Lier les applications externes au local
1. Sélectionner l'application dans la bibliothèque Applications externes
2. Cliquer sur « Lier au local »
![](https://file.shimoko.com/d/openlist/openlist/%E7%BD%91%E7%AB%99%E5%AA%92%E4%BD%93%E6%96%87%E4%BB%B6/linkback.gif?sign=3xdVe7IFYcMfwNt1K9WMk8ZQYIlNMWngjZoTkNwA610=:0)

## Supprimer le lien
Cliquer sur le bouton Supprimer le lien à droite dans la liste des applications liées.
![](https://file.shimoko.com/d/openlist/openlist/%E7%BD%91%E7%AB%99%E5%AA%92%E4%BD%93%E6%96%87%E4%BB%B6/jiechu.gif?sign=CLDMEQxHts1S2mBEnka9HUYT2XMzcnBwX1Sjfas9Dho=:0)

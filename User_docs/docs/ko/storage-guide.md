# 외장 저장소 가이드

## 권장 구성

| 구성 | 권장값 | 설명 |
|------|--------|------|
| 용량 | 256 GB 이상 | 마이그레이션된 앱 수에 따라 다름 |
| 인터페이스 | USB 3.0 이상 / Thunderbolt | USB 2.0은 느림; 대용량 앱 마이그레이션에 더 많은 시간 소요 |
| 파일 시스템 | APFS | 클론, 스냅샷 지원, 최상의 성능 |

## 인터페이스 성능 비교

| 인터페이스 | 이론 속도 | 실제 마이그레이션 속도 | 사용 사례 |
|-----------|---------|---------------------|----------|
| USB 2.0 | 480 Mbps | ~30 MB/s | 비권장; 너무 느림 |
| USB 3.0 (USB-A) | 5 Gbps | ~350 MB/s | 기본적으로 충분 |
| USB 3.1 Gen 2 (USB-C) | 10 Gbps | ~700 MB/s | 권장 |
| Thunderbolt 3/4 | 40 Gbps | ~2500 MB/s | 최상의 성능 |
| NVMe (Thunderbolt) | 40 Gbps | ~2800 MB/s | 최상의 성능 |

## 파일 시스템 권장 사항

### APFS (권장)

- 클론, 스냅샷, 공간 공유 지원
- 최상의 성능, 특히 SSD에 적합
- macOS 네이티브 지원

### HFS+

- 호환성 양호; 구형 Mac에 적합
- 클론과 스냅샷 미지원
- 기계식 하드 드라이브에 적합

### exFAT

- 크로스 플랫폼 호환 (macOS + Windows)
- 하드 링크와 클론 미지원
- 상대적으로 낮은 성능
- 여러 시스템에서 사용해야 하는 시나리오에 적합

## 용량 계획

마이그레이션 후 AppPorts의 외장 저장소 사용량은 마이그레이션된 앱과 데이터 디렉토리의 크기에 따라 달라집니다. 아래는 일반적인 앱의 참조 크기입니다:

| 앱 유형 | 크기 |
|---------|------|
| Chrome | ~500 MB |
| Microsoft Office | ~5 GB |
| Adobe Creative Cloud | ~20-50 GB |
| Xcode | ~15 GB |
| Final Cut Pro | ~5 GB |
| 로컬 대규모 언어 모델 (Ollama) | ~4-30 GB |

::: tip 💡 용량 권장 사항
- 가벼운 사용 (5-10개 앱): 128 GB
- 보통 사용 (10-20개 앱): 256 GB
- 무거운 사용 (20개 이상 앱 + 데이터 디렉토리): 512 GB 이상
:::

## 참고 사항

- 외장 저장소는 연결된 상태를 유지해야 합니다; 마이그레이션된 앱과 데이터 디렉토리는 오프라인에서 사용할 수 없습니다
- 외장 저장소의 데이터를 정기적으로 백업하세요
- 마이그레이션 중에 외장 저장소를 분리하지 마세요
- 외장 저장소에 장애가 발생하면 AppPorts를 통해 앱을 로컬로 다시 이동할 수 있습니다

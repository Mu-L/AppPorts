# 호환성 및 제한 사항

## 시스템 요구 사항

| 요구 사항 | 설명 |
|----------|------|
| 최소 OS 버전 | macOS 12.0 (Monterey) |
| 아키텍처 | Intel x86_64 / Apple Silicon (arm64) |
| 권한 | 전체 디스크 접근 |
| 외장 저장소 | 최소 하나의 외장 저장소 장치 필요 |

## 기능 호환성

### macOS 버전별

| 기능 | macOS 12.0 - 15.0 | macOS 15.1+ |
|------|:---:|:---:|
| 앱 마이그레이션 (Stub Portal) | ✓ | ✓ |
| 데이터 디렉토리 마이그레이션 | ✓ | ✓ |
| 코드 서명 관리 | ✓ | ✓ |
| App Store 앱 외장 드라이브 마이그레이션 | ✗ | ✓ |
| App Store 앱 외장 드라이브 제자리 업데이트 | ✗ | ✓ |
| iOS 앱 마이그레이션 | ✓ | ✓ |

::: warning ⚠️ macOS 15.1 미만의 App Store 앱
macOS 15.1 (Sequoia) 이전 버전은 App Store 앱의 외장 드라이브 설치를 지원하지 않습니다. AppPorts 설정에서 "App Store 앱 마이그레이션"을 수동으로 활성화해야 하며, 앱 업데이트 시 수동으로 재마이그레이션하여 덮어써야 합니다.
:::

### 앱 유형별

| 앱 유형 | 마이그레이션 | 복원 | 자동 업데이트 | 비고 |
|---------|:---:|:---:|:---:|------|
| 네이티브 macOS 앱 | ✓ | ✓ | ✓ | 최상의 호환성 |
| Sparkle 앱 | ✓ | ✓ | 잠금 필요 | 잠금이 앱 내부 업데이트를 방지; 업데이트하려면 복원해야 함 |
| Electron 앱 | ✓ | ✓ | 잠금 필요 | Sparkle와 동일 |
| Chrome / Edge (커스텀 업데이터) | ✓ | ✓ | ✓ | 업데이터가 로컬에 설치; 외부 복사본을 손상시키지 않음 |
| App Store 앱 (macOS 15.1+) | ✓ | ✓ | ✓ | 네이티브 외장 설치; App Store가 직접 업데이트 가능 |
| App Store 앱 (macOS <15.1) | ✓ | ✓ | 수동 | 업데이트 시 재마이그레이션 필요 |
| iOS 앱 (Mac 버전) | ✓ | ✓ | ✓ | iOS Stub Portal 사용 |
| 시스템 앱 | ✗ | — | — | SIP 보호; 마이그레이션 불가 |

### 데이터 디렉토리 유형별

| 데이터 디렉토리 유형 | 마이그레이션 | 위험 |
|--------------------|:---:|------|
| `~/Library/Application Support/` | ✓ | 중간 — 파일 잠금 또는 SQLite WAL 로그를 사용할 수 있음 |
| `~/Library/Preferences/` | ✓ | 낮음~중간 — `cfprefsd` 캐싱이 오래된 읽기를 유발할 수 있음 |
| `~/Library/Containers/` | ✓ | 중간 — 동일 Team 하위의 앱이 공유 |
| `~/Library/Group Containers/` | ✓ | 중간 — 공유 데이터가 다른 앱을 방해할 수 있음 |
| `~/Library/Caches/` | ✓ | 낮음 — 캐시는 재구성 가능 |
| `~/Library/Logs/` | ✓ | 낮음 — 로그 파일만 |
| `~/Library/WebKit/` | ✓ | 중간 — WebKit 로컬 저장소 |
| `~/Library/HTTPStorages/` | ✓ | 낮음 — 네트워크 세션 저장소 |
| `~/Library/Application Scripts/` | ✓ | 낮음 — 확장 스크립트 |
| `~/Library/Saved Application State/` | ✓ | 낮음 — 윈도우 상태 복원 |
| `~/.npm`, `~/.m2` 등 dot-folder | ✓ | 낮음 — 개발 도구 캐시 |

## 마이그레이션 불가능한 콘텐츠

### SIP 보호

| 경로 | 이유 |
|------|------|
| macOS 시스템 앱 (Safari, Finder 등) | 시스템 무결성 보호 |
| `~/Library/Containers/` 최상위 디렉토리 | macOS 시스템 보호 |

### 경로 참조 포함

| 경로 | 이유 |
|------|------|
| `~/.local` | 실행 경로 참조를 포함; 마이그레이션 후 커맨드라인 도구가 실패할 수 있음 |
| `~/.config` | 절대 경로 설정을 포함; 마이그레이션 후 도구 설정이 실패할 수 있음 |

## 외장 저장소 요구 사항

| 요구 사항 | 설명 |
|----------|------|
| 파일 시스템 | APFS, HFS+, exFAT 지원 |
| 최소 공간 | 마이그레이션된 앱 크기에 따라 다름 |
| 인터페이스 | USB, Thunderbolt, NVMe 모두 지원 |
| 연결 유지 | 마이그레이션 후 외장 저장소가 연결된 상태를 유지해야 함; 그렇지 않으면 앱 실행 불가 |

::: tip 💡 파일 시스템 권장 사항
- **APFS**: 권장; 클론, 스냅샷 지원, 최상의 성능
- **HFS+**: 호환성 양호; 구형 Mac에 적합
- **exFAT**: 크로스 플랫폼 호환; 하드 링크와 클론 미지원
:::
